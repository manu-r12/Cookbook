# AI Recipe Assistant

Welcome to the AI Recipe Assistant project! This application helps users find recipes based on the ingredients they have at hand. It leverages a Core ML model to detect vegetables, uses the Spoonacular API to fetch recipes, and incorporates Firebase for user data management. The user interface is built with SwiftUI. This README provides an overview of the project setup, dependencies, and usage.

## Project Overview

AI Recipe Assistant is an iOS application that assists users in discovering recipes based on the ingredients they have. Users can save their favorite recipes and share them with others. The app includes features like ingredient detection using a Core ML model, recipe fetching from the Spoonacular API, and user data management with Firebase.

## Features

- **Ingredient Detection**: Use the Core ML model to detect vegetables from images.
- **Recipe Search**: Fetch recipes based on provided ingredients using the Spoonacular API.
- **User Authentication**: Sign up, log in, and manage user profiles using Firebase Authentication.
- **Recipe Management**: Save and share recipes.
- **Modern UI**: Built with SwiftUI for a sleek and responsive user experience.
